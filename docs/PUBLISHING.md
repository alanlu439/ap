## Enable GitHub Pages

If the site returns 404, enable Pages manually:

1. Open repository Settings.
2. Open Pages.
3. Set Source to `Deploy from a branch`.
4. Select branch `main` and folder `/root`.
5. Save and wait a few minutes.
